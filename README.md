# Поднятие pfSense для локальной сети

## О проекте

Данный проект посвящён развёртыванию и базовой настройке pfSense для организации и защиты локальной сети.

В ходе работы была выполнена:

- подготовка среды для установки pfSense;
- установка операционной системы pfSense;
- настройка сетевых интерфейсов;
- конфигурация DHCP, DNS и NAT;
- изучение веб-интерфейса системы;
- диагностика сети;
- настройка маршрутизации и VPN;
- резервное копирование конфигурации.

Проект демонстрирует основные возможности pfSense как программного маршрутизатора и межсетевого экрана.

---

# Что такое pfSense

pfSense — это специализированная операционная система на базе FreeBSD, предназначенная для:

- маршрутизации сетевого трафика;
- организации локальных сетей;
- защиты сети с помощью Firewall;
- настройки VPN;
- управления сетевыми службами.

Система широко используется как в домашних сетях, так и в корпоративной инфраструктуре благодаря:

- высокой стабильности;
- гибкости настройки;
- открытому исходному коду;
- удобному веб-интерфейсу;
- большому количеству встроенных функций.

Официальный сайт проекта:

https://www.pfsense.org/

# Подготовка среды

Перед установкой pfSense необходимо подготовить окружение.

## Создание виртуальной машины

Для тестирования pfSense рекомендуется использовать виртуальную машину.

### Основные параметры VM:

| Параметр | Значение |
|---|---|
| Тип ОС | FreeBSD (64-bit) |
| RAM | 2048 MB |
| CPU | 2 ядра |
| Диск | 10–20 GB |

## Настройка сетевых адаптеров

Необходимо создать минимум два сетевых интерфейса:

### WAN

Используется для подключения к интернету.

### LAN

Используется для локальной сети.

Пример:

| Интерфейс | Назначение |
|---|---|
| Adapter 1 | WAN |
| Adapter 2 | LAN |

# Установка pfSense

## Шаг 1. Загрузка ISO-образа

Скачать образ можно с официального сайта:

https://www.pfsense.org/download/

## Шаг 2. Подключение ISO

Подключить ISO-файл к виртуальной машине.

## Шаг 3. Запуск установки

После запуска выбрать:

- Install pfSense
- Accept
- Auto (ZFS или UFS)

## Шаг 4. Завершение установки

После завершения:

1. Перезагрузить систему.
2. Извлечь ISO.
3. Запустить pfSense.

---

# Первичная настройка

После запуска система предложит назначить сетевые интерфейсы.

## Назначение интерфейсов

Пример:

| Интерфейс | Назначение |
|---|---|
| em0 | WAN |
| em1 | LAN |

## Настройка LAN

Пример IP:

```text
192.168.1.1/24
```

После этого веб-интерфейс будет доступен по адресу:

```text
https://192.168.1.1
```

---

# Вход в веб-интерфейс

## Данные по умолчанию

| Параметр | Значение |
|---|---|
| Логин | admin |
| Пароль | pfsense |

После входа рекомендуется сразу изменить пароль администратора.

---

# Настройка DHCP

## Включение DHCP-сервера

Путь:

```text
Services → DHCP Server
```

## Пример диапазона адресов

```text
192.168.1.100 - 192.168.1.200
```

DHCP позволяет автоматически выдавать IP-адреса устройствам локальной сети.

---

# Настройка NAT

NAT используется для доступа устройств локальной сети в интернет.

По умолчанию pfSense автоматически создаёт правила NAT.

Путь:

```text
Firewall → NAT
```

---

# Настройка Firewall

pfSense содержит мощный межсетевой экран.

## Основные функции Firewall

- фильтрация трафика;
- блокировка подозрительных подключений;
- контроль доступа;
- защита локальной сети.

## Путь к правилам

```text
Firewall → Rules
```

---

# Диагностика системы

pfSense содержит встроенные инструменты диагностики.

## Доступные инструменты

### Ping

Проверка доступности узлов.

### Traceroute

Анализ маршрута пакетов.

### System Logs

Просмотр системных журналов.

### Packet Capture

Анализ сетевого трафика.

# VPN

pfSense поддерживает:

- OpenVPN;
- IPsec;
- WireGuard.

VPN используется для:

- удалённого доступа;
- безопасного соединения;
- объединения сетей.
